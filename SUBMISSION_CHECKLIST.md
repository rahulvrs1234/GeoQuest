# Submission Checklist

Use this checklist before uploading anything.

## Must Submit

- Android Studio project source code
- GitHub repository link, if your unit requires it
- `README.md`
- 1000-word reflection using Gibbs Reflective Cycle
- any screenshots, PDF, or report files specifically requested by your lecturer

## Included in This Project

- `README.md`
- `ASSESSMENT_MAPPING.md`
- `GIBBS_REFLECTION.md`
- full Android app source code

## App Requirements Check

- Kotlin app in Android Studio
- Jetpack Compose UI
- Material Design 3
- Home screen
- Quiz activity screen
- Settings screen
- Statistics screen
- navigation
- networking
- local database
- DataStore preferences
- WorkManager
- notification permission
- tests

## Final Proof Check

- app opens successfully on a device or emulator
- home, quiz, stats, and settings screens all work
- reminder toggle works correctly
- theme switching works correctly
- hint button works correctly in quiz rounds
- custom app icon appears correctly after install
- no obvious crashes during normal use

## Before Uploading

- add your name, student ID, unit code, and tutor details where required
- read `GIBBS_REFLECTION.md` once in your own voice and adjust wording if needed
- convert the reflection to PDF or Word if the LMS requires that format
- make sure screenshots are clear and show the final version of the app
- if uploading source as a zip, exclude `.gradle`, `.idea`, `build`, and `local.properties`

## Suggested Screenshot Set

- launcher icon on phone
- home screen
- quiz screen before answering
- quiz screen after answering
- statistics screen
- settings screen
